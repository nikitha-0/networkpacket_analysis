# Network Traffic Threat Detection with Suspicious IP Detection
This project analyzes network traffic to identify suspicious IP addresses based on abnormal communication frequency. Network packets are captured using Wireshark and exported as pcap files. These packet traces are then processed using Python to extract patterns.
This project demonstrates practical implementation of network traffic analysis,intrusion detection concepts, and cyber-threat intelligence enrichment.
SYSTEM WORKFLOW: 
1. Capture network traffic using Wireshark
2. Export packets as PCAP file
3. Load PCAP into Python analyzer
4. Extract source/destination IPs
5. Count packet frequency per IP
6. Detect IPs exceeeding threshold

TECHNOLOGIES USED:
1. Python
2. Wireshark
3. Networking concepts(IP,Packets,traffic flow)
